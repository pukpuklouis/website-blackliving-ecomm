# Icons Directory

This directory contains SVG icons used in the CircularAdvantages component on the About page.

## Required Icons

Please add the following SVG icon files to this directory:

1. **plane.svg** - Airplane icon for "產地直營進口" (Direct Import)
2. **warehouse.svg** - Warehouse icon for "自有倉儲管理" (Own Warehouse)
3. **shopping-cart.svg** - Shopping cart icon for "優質購物體驗" (Shopping Experience)
4. **chat.svg** - Chat/messaging icon for "即時客戶服務" (Customer Service)

## Specifications

- **Format**: SVG
- **Size**: 64x64px (or scalable)
- **Style**: Simple, clean line icons
- **Color**: Preferably single color (will be styled with CSS if needed)

## Usage

These icons are used in:
- `/apps/web/src/components/CircularAdvantages.astro`

Icons are displayed at 64x64px size using Astro's `<Image>` component.
